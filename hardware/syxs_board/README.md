## 实物调试记录（截至 2026-07-20）

- 接收反相放大链路的 `R16` 在距离测试中先调整为 `130 kΩ`，后进一步调整至 `270 kΩ`；测量板上实装值后再计算实际增益。
- 第二版测试板为解决启动瞬间输入掉压，将 `C3` 更换为 `10 µF` 后恢复正常开机。
- `C32` 曾在电源排障中暂时拆除；是否恢复 `47 µF` 应以 `5V_IN/5V_SYS` 启动波形和多次冷启动结果为准。
- 第三块板仍需重点复查 VCAP、R34、运放中点偏置以及接收滤波器电阻电容焊点；这些是待验证项，不是已经确认的设计错误。
