# STM32 发送端工程

这是 STM32F411CEU6 黑丸调试工程，用于通过串口命令产生 4-FSK 测试音和一位数字帧。

## 当前调试引脚

- PWM 输出：`PA8 / TIM1_CH1`。
- PWM 更新：`TIM2` 中断按音频采样率推进 DDS 相位。
- 串口：`USART2`，115200 8N1。

整板原理图中发送链路已经改到 `PA0`。如果烧录到整板，需要在 CubeMX/代码中把 PWM 输出从 `PA8/TIM1_CH1` 改到 `PA0` 对应的定时器通道。

## 串口命令

- 输入 `00`：输出 1500 Hz。
- 输入 `01`：输出 2500 Hz。
- 输入 `10`：输出 3500 Hz。
- 输入 `11`：输出 4500 Hz。
- 输入 `s`：停止连续测试音。
- 输入 `0` 到 `9`：发送一位数字帧。

## 工程依赖

为避免仓库内重复提交两份 HAL/CMSIS，Keil 工程中的驱动路径已改为复用：

`software/stm32_receive/Drivers`

因此构建发送端前，需要保证接收端工程目录仍保留完整 `Drivers`。
