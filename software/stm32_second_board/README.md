# 第二版板卡三模式收发工程

这是当前“声语信使”主固件，源自本地工程 `D:\stm32\test2`，运行于 STM32F411RET6 第二版收发一体板。

## 主要功能

- 半双工 4-FSK 收发：接收端为 `PC4 / ADC1_IN14 + MCP6S21`，发送端为 `MCP4921 DAC + PAM8302`。
- 三种模式：STANDARD、MULTI-NODE、HIDDEN；开机后用 `A/B/C` 选择。
- 普通频点：1500/2500/3500/4500 Hz，同步频点 2000 Hz，ADC 采样率 16 kHz。
- 高频频点：6400/6550/6850/7000 Hz，同步频点 6700 Hz，ADC 采样率 20 kHz。
- 固定 48 字符负载，CRC12 多项式 `0x80F`，两块 RS(35,25) GF(64) 前向纠错。
- 每个 6 bit RS 码元使用 3 个 4-FSK 符号，并插入同步标记；完整帧 289 个发送步长，约 19.941 秒。
- STANDARD/HIDDEN 支持两遍校准及 PGA 自动增益；MULTI-NODE 不校准并固定 PGA x2。
- 多机模式支持 N1/N2/N3 和广播目的地址，地址参与 CRC 校验。
- Flash 保存最近 5 条消息；LED3 指示发送，蓝灯指示接收完成。
- OLED 开机画面显示“声语信使”以及成员姓名、学号 2 秒。

## 操作概要

1. 上电后等待 2 秒开机画面结束。
2. 双方按相同模式键：`A` 标准、`B` 多机、`C` 高频隐蔽。
3. STANDARD/HIDDEN 首次进入发送状态后按 `#` 发送校准序列，再按 `*` 返回接收；MULTI-NODE 无校准。
4. 在发送状态使用数字键多击输入，`A/B` 移动光标，`C` 删除，`D` 清空，`#` 发送。
5. 在接收状态使用 `A/B/C` 查看较旧、较新或最新的已存消息，按 `*` 切换收发。
6. 多机模式在接收状态按 `1/2/3` 设置本机节点，按 `D` 切换目的节点或广播。
